# Data Analytics Portfolio

This is a hands on data analytics learning portfolio focused on
data workflows using Python, SQL, and Power BI.

## Tools and Technologies

- Python 3.x
- Pandas, NumPy, Matplotlib, Seaborn
- PostgreSQL
- Power BI
- VS Code with Jupyter extension

## Environment Setup

### 1. Clone the repository

git clone <repo url>

### 2. Create virtual environment

python -m venv venv

### 3. Activate virtual environment

# Windows

venv\Scripts\activate

### 4. Install dependencies

pip install -r requirements.txt

### 5. Select interpreter in VS Code

- Press Ctrl + Shift + P
- Python: Select Interpreter
- Choose the venv option

## Projects

Each folder contains a dataset and notebook with full analysis
from raw messy data to final insights.
